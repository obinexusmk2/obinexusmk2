Technical Infrastructure

The OBIX SDK

50 packages. Zero React dependency. Pure OBIX runtime with libpolycall-v1 telemetry. Constitutional computing, data-oriented, accessibility-first. 

# Install the OBIX runtime
npm install @obinexusltd/obix-core
npm install @obinexusltd/obix-cli --save-dev

# Build your OBIX project
npx @obinexusltd/obix-cli build --target esm,cjs --out dist

# Validate JSON schema
npx @obinexusltd/obix-cli validate ./schema/component.json

# Hot-swap development
npx @obinexusltd/obix-cli hot-swap --watch src --delay 500
